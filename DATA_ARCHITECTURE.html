Data Flow Diagrams

User Registration

User → Frontend → Backend API → MongoDB (users collection) → JWT token returned

Asking a Question

User → Frontend → Backend API → MongoDB (questions collection)
                          → Background: AI generates answer via Groq API
                          → MongoDB (answers collection, isAIGenerated: true)

AI Chat

User → Frontend → Backend API → Groq API → Response with real shlokas from vedabase.io
                          → MongoDB (aichats collection stores history)

Guru Verification

Guru → Guru Portal → Backend API → MongoDB (answers.isVerifiedByAdmin: true)
                              → Answer now visible to all users
